## 6. FAQ para Comunicacao

### Perguntas Frequentes

**P: O que e o Neoloop Design System Builder?**
R: E uma plataforma visual para criar e gerenciar design systems, incluindo design tokens, componentes UI e ferramentas de validacao de acessibilidade.

**P: Quanto custa?**
R: [Definir pricing - se gratuito, mencionar. Se pago, detalhar planos]

**P: Preciso saber programar?**
R: Nao! A interface e totalmente visual. Desenvolvedores podem exportar tokens e codigo gerado.

**P: Como funciona a validacao WCAG?**
R: O Neoloop inclui 3 validadores integrados: Contrast Checker (cores), Typography Validator (fontes) e Spacing Validator (espacamento). Todos seguem as diretrizes WCAG 2.1.

**P: Quais navegadores sao suportados?**
R: Chrome 90+, Firefox 88+, Safari 14+ e Edge 90+.

**P: Posso exportar meu design system?**
R: Sim! Na v1.0, suportamos export em JSON. Nas proximas versoes, adicionaremos CSS, SCSS e integracao com Figma.

**P: Existe suporte?**
R: Sim! Documentacao completa disponivel em [link]. Para questoes especificas, abra uma issue em [link] ou envie email para [email].
